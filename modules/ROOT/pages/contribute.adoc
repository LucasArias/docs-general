= Contribute to MuleSoft Documentation
ifndef::env-site,env-github[]
include::_attributes.adoc[]
endif::[]

Many of our documentation repositories are open and we welcome your contributions!

To contribute to the English version of this documentation:

. Sign the https://cla.salesforce.com/sign-cla[Salesforce.com Contributor License Agreement (CLA)] if this is your first time contributing. The CLA only needs to be signed once per GitHub account.
. Click *Edit in Github* on the page you want to modify.
. Make your edits.
. Submit a pull request to the Git repository.

If you submit the pull request without signing the CLA first, you will receive an error in the pull request. Sign the CLA to remove this error.


The Documentation team will either accept your pull request or reach out if there are questions.
